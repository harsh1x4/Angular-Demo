Angular Commands List

New Command

To create a new Project you have to use this command.
ng new <ProjectName>

Example :- If you want to create a project named appdemo
ng new <appdemo>

Generate Command

It will give a list of Schematic which you can generate.
ng generate
Or
ng g

Available Schematics: Collection "@schematics/angular" (default):
  • app-shell
  • application
  • class
  • component
  • directive
  • enum
  • guard
  • interceptor
  • interface
  • library
  • module
  • pipe
  • resolver
  • service
  • service-worker
  • web-worker

Generate Component

Example :- If you want to generate a component name home
ng generate component <home>

To start a project

There are two commands to start a project:
The best one is
--npm start
Second one is
--ng serve
